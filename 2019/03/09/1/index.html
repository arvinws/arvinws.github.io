初始化一个vue项目脚手架 并且引入ElementUI

初始化一个vue项目脚手架

cnpm install vue

全局安装 vue-cli

$ cnpm install –global vue-cli

创建一个基于 webpack 模板的新项目

$ vue init webpack my-project

下一步 需要进行一些配置，默认回车即可

配置完成后

$ cd my-project
$ cnpm install
$ cnpm run dev

引用element

cnpm i element-ui s
